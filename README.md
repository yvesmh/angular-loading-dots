# loading-dots

Angular directive for displaying loading dots in the form of text

## Getting Started

Refer to our [Getting started guide](http://yvesmh.github.io/angular-loading-dots/)

## Building and testing

Requisites:

1. NodeJS and npm
2. Grunt

### Building

To build just run:

    grunt build

### Testing

To run the tests:

    grunt test
